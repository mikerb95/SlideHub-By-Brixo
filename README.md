# SlideHub-By-Brixo
SlideHub — Sistema de control de presentaciones en tiempo real Microservicios Spring Boot (Java 21) con sincronización multi-pantalla, notas del presentador asistidas por IA (Gemini + Groq), integración con repositorios GitHub y Deploy Tutor. Stack: Spring Security OAuth2, Redis, MongoDB, PostgreSQL, Thymeleaf, S3, despliegue en Render.
