Feature Matrix

Each capability is a discrete module within the layered architecture, designed for independent operation and composability.

SYNC_PROTOCOL

Multi-Screen Synchronization

Projector, demo, presenter, and remote views — all synchronized in real-time via HTTP polling through the gateway. Slide state lives in Redis with sub-second propagation.

poll_interval: 1000ms · protocol: HTTP/REST · store: Redis · views: 5
HAPTIC_REMOTE

Touch Remote

Swipe navigation with vibration feedback. Push-to-talk voice assistant via Groq Whisper transcription.

input: swipe + voice · haptic: vibration API
AI_PIPELINE

AI Notes

Gemini Vision analyzes slide PNGs. Groq generates structured presenter notes with key phrases and demo tags.

vision: Gemini · gen: Groq · store: MongoDB
DUAL_MODE

Demo Mode

Switch between slides and live URL iframes from the master panel. Return-slide memory preserves context.

modes: "slides" | "url" · returnSlide: auto
CONTROL_SURFACE

Master Panel

Tablet-optimized grid with slide thumbnails, meeting session management, participant assignments with haptic feedback, and a quick-action sidebar.

viewport: 11" tablet · grid: auto-fill 160px · auth: PRESENTER | ADMIN
DEPLOY_TUTOR
